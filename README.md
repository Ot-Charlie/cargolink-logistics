# CargoLink Logistics

🌐 **Live Demo:** [cargolink-logistics.vercel.app](https://cargolink-logistics.vercel.app)

A modern logistics web platform connecting Africa to global markets with real-time shipment tracking, schedule booking, and complete visibility from port to doorstep.

## Tech Stack

- **Framework:** Next.js (App Router) with TypeScript

- **Styling:** Tailwind CSS v4

- **Fonts:** Urbanist, Inter, DM Sans (via next/font/google)

## Features

- Real-time shipment tracking

- Schedule booking with transport mode selection

- Responsive design (mobile-first)

- Glassmorphism UI

- Scroll-triggered animations

## Getting Started

```bash

npm run dev

```

Open [http://localhost:3000](http://localhost:3000) to view in your browser.

## Deployment

Deployed on [Vercel](https://vercel.com). Every push to main triggers an automatic redeployment.
